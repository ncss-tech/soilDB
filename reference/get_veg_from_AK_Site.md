# Get Vegetation Data from an AK Site Database

Get Vegetation Data from an AK Site Database

## Usage

``` r
get_veg_from_AK_Site(dsn)
```

## Arguments

- dsn:

  file path the the AK Site access database

## Value

A data.frame with vegetation data in long format, linked to site ID.
