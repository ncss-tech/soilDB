# Get Soil Color Data from a PedonPC Database

Get, format, mix, and return color data from a PedonPC database.

## Usage

``` r
get_colors_from_pedon_db(dsn)
```

## Arguments

- dsn:

  The path to a 'pedon.mdb' database.

## Value

A data.frame with the results.
