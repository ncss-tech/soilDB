# Get Site Data from a PedonPC Database

Get site-level data from a PedonPC database.

## Usage

``` r
get_site_data_from_pedon_db(dsn)
```

## Arguments

- dsn:

  The path to a 'pedon.mdb' database.

## Value

A data.frame.
