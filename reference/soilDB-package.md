# Soil Database Interface

A collection of functions for reading soil data from U.S. Department of
Agriculture Natural Resources Conservation Service (USDA-NRCS) and
National Cooperative Soil Survey (NCSS) databases

## Details

This package provides methods for extracting soils information from
local NASIS databases (MS SQL Server), local PedonPC and AKSite
databases (MS Access format), Soil Data Access, and other soil-related
web services.
